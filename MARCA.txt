                                  Uso: logo del nav, favicon, Apple Touch Icon,
                                  íconos del manifest.json.

                                  Uso: documentos, presentaciones, email.
                                  No está en el HTML actualmente.

    carrete.png                ← [PENDIENTE — entregar a 512×512 px, fondo
                                  transparente, 72 dpi]
                                  Para activar:
                                    1. Copiar el archivo a assets/carrete.png
                                    2. El sitio lo detecta automáticamente y

────────────────────────────────────────────────────────────────────────────────
  BILINGÜISMO
────────────────────────────────────────────────────────────────────────────────

  El sitio tiene español e inglés. El idioma se controla con un objeto JS
  al final del index.html, justo antes del cierre </body>.

  Cómo funciona:
    — Todos los textos del HTML tienen data-t="clave"
    — El objeto T.es y T.en contienen los textos por clave
    — El botón de idioma en el header alterna entre los dos
    — Idioma por defecto: español (es)
    — El atributo <html lang> se actualiza automáticamente al cambiar

  Para cambiar un texto:
    1. Buscar la clave data-t en el HTML
    2. Editar el texto en T.es (español) y T.en (inglés) al final del archivo
    — No editar el texto directamente en el HTML o se desincroniza

────────────────────────────────────────────────────────────────────────────────
  WORKER DE LEADS
────────────────────────────────────────────────────────────────────────────────

  Archivo:  workers/leads.js
  Config:   workers/wrangler.toml (NO en GitHub — tiene secretos)

  Para activar:
    1. Cuenta gratuita en Cloudflare Workers
    2. Crear KV namespace "TURICINE_LEADS"
    3. Pegar el ID del KV en wrangler.toml → id = "..."
    4. Definir ADMIN_SECRET en el dashboard de Cloudflare (no en el .toml)
    5. Deploy: cd workers && npx wrangler deploy
    6. Copiar la URL del Worker desplegado al listener del formulario

  Para ver los leads:
    GET https://TU-WORKER.workers.dev/leads?secret=TU_CLAVE
